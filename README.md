# 🌤️ Weather Forecast Web Application

A responsive and user-friendly weather application that provides real-time weather information for any city using the OpenWeather API.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature display
- ☁️ Weather condition information
- 💧 Humidity details
- 🌬️ Wind speed information
- 🌅 Sunrise time display
- 🌇 Sunset time display
- 📱 Responsive design for all devices
- ⚡ Fast and accurate weather updates

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## 🚀 Getting Started

### Prerequisites

- A web browser
- OpenWeather API Key

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Add your OpenWeather API key in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

4. Open `index.html` in your browser.
